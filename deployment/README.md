# Wellness Tourism Package Prediction App

This Streamlit application predicts the likelihood of a customer purchasing a Wellness Tourism Package based on their profile and interaction data.

## Features
- Real-time prediction using machine learning model
- Interactive user interface with Streamlit
- Model and encoders hosted on Hugging Face Hub

## Model
The model is trained on tourism customer data and hosted at: `Abhik19/tourism-prediction-model`

## How to Use
1. Enter customer details in the input fields
2. Click "Predict Purchase Likelihood"
3. View the prediction results and probability

## Technical Stack
- Streamlit for UI
- scikit-learn for ML model
- Hugging Face Hub for model storage
- Docker for deployment
